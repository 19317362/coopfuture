Detailed Description

template<typename ErrorType>
class fut::Future< void, ErrorType >

A class to encapsulate a result-less asynchronous task.

This is a specialization of the Future class. It is used as a form of barrier, where the Future will not have a result set on it but can throw an error and can be blocked on.

This class is NOT thread-safe and is intended for cooperative multi-tasking.
