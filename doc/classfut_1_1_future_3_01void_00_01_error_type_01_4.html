Detailed Description

template<typename ErrorType>
class fut::Future< void, ErrorType >

A class to encapsulate a resultless asynchronous task.

This is a specialization of the Future class. It is used as a form of barrier, where the Future will not have a result set on it but can throw an error and can be blocked on.

Returns the underyling result/exception, blocking if necessary.

If the Future has completed then throw the error or return silently as appropriate, otherwise, add a waiting context node to the cons list and jump to the scheduler (block until Future is resolved).
